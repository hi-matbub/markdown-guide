# Code and Syntax Highlighting

Used for sharing bits of code. There are several approaches with GitHub. 

## 1. Code Block

```javascript
const hello = (name) => {
  return 'Hello, ' + name + '!';
};

hello('World');
```

## 2. Syntax Highlight

`console.log('Hello, World!');`
